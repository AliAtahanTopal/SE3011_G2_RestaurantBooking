-------------------------
LOCAL INSTALLATION:
-------------------------
1-Download XAMPP link: https://www.apachefriends.org/tr/index.html
2-Download MySQL Workbench link: https://www.mysql.com/products/workbench/
3-Run XAMPP
4-Start MySQL and Apache from XAMPP
5-Run MySQL Workbench
6-Create a new schema named "rbsdb" via MySQL Workbench.
7-Clone github repository
8-Copy 'createtables.sql' and 'insertdata.sql' queries from "..\Github\SE3011_G2_RestaurantBooking\DOCS\Implementation\db"
9-Paste them into MySQL Workbench and run the queries in that order.
10-Copy Project folder under "..\Github\SE_3011_G2_RestaurantBooking\DOCS\Implementation\"
11-Paste it into "..\xampp\htdocs"
12-Rename the folder as RestaurantBookingSystem
13-Start Apache via XAMPP
14-Paste "http://localhost/RestaurantBookingSystem/" into your browser's address bar. (We recommend Google Chrome)
